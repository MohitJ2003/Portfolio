Cybersecurity & Ethical Hacking using Termux

Setup Commands

  • Update & Upgrade
    • pkg update && pkg upgrade
  • Install Git
    • pkg install git
  • Install Python
    • pkg install python
  • Install Nmap
    • pkg install nmap
  • Install Curl
    • pkg install curl

Tools Usage

  • Nmap Scan
    • nmap -sV <target-ip>
  • Install Nikto
    • pkg install nikto
  • Run Nikto
    • nikto -h http://example.com
  • Install Hydra
    • pkg install hydra
  • Install theHarvester
    • git clone https://github.com/laramies/theHarvester
  • Run theHarvester
    • python3 theHarvester.py -d example.com -b google

Learning Roadmap

  • Linux Basics
  • Networking (IP, DNS, Ports)
  • Web Basics (HTTP, HTML, JS)
  • Cyber Attacks (SQLi, XSS, Brute Force)
  • Python Scripting

Important Notes

  • Use tools only on your own systems
  • Practice on legal platforms like TryHackMe
  • Unauthorized hacking is illegal
